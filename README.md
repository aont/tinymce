# TinyMCE demo

This is a repository for a simple demo page using TinyMCE.

## Update TinyMCE

Please download the updated version of TinyMCE from these pages.

- [Self-hosted releases](https://www.tiny.cloud/get-tiny/self-hosted/)
- [Language Packages](https://www.tiny.cloud/get-tiny/language-packages/)
